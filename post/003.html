Firefox for Android 信任系统第三方证书

发表于2025-07-28|更新于2025-09-03|Android
|字数总计:78|阅读时长:1分钟|阅读量:|评论数:

Android 版本的 Firefox 默认不信任系统证书需要按以下步骤开启信任系统证书

步骤

  1. 打开 Firefox for Android
  2. 打开设置-关于 Firefox
  3. 点击七次 Firefox 的 logo 图标解锁 Secret Setting
  4. 返回，然后找到 Secret Setting，开启 Use third party CA certificates
  5. 完成，最好重启 Firefox
